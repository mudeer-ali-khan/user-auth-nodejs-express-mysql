## Project Description

This repository contains **Lab 10** for the **Web Engineering / Web Development** course. The project implements a complete **user registration and authentication system** using **Node.js**, **Express.js**, **MySQL (MySQL Workbench)**, **HTML**, **CSS**, and **JavaScript**.

The application allows users to:

* Register using a web form
* Store user data in a MySQL database
* Log in using email and password
* Authenticate credentials using SQL queries
* Access a dashboard after successful login

This lab demonstrates practical usage of **server-side routing, middleware, database connectivity, and form handling**.
